> **Proyek Pengembangan Aplikasi Jamu Bae** oleh Pradja Agung Kurniawan
>
> JamuBae adalah solusi inovatif untuk memodernisasi dan mempermudah akses masyarakat terhadap Usaha Mikro, Kecil, dan Menengah (UMKM) Jamu di Indonesia. Aplikasi ini berperan sebagai platform penghubung digital yang menyajikan berbagai varian jamu dari penjual terpercaya langsung ke tangan konsumen.

---

## ğŸ¯ Alasan dan Tujuan Pembuatan

Pengembangan aplikasi JamuBae didasarkan pada tiga alasan utama:

### 1. Digitalisasi UMKM
* **Tantangan:** Sebagian besar UMKM Jamu masih beroperasi secara konvensional, mengandalkan penjualan fisik atau dari mulut ke mulut, membuat mereka rentan terhadap perubahan pasar dan kurang terlihat (*visible*) di mata konsumen modern.
* **Solusi:** Memberikan *platform* digital untuk membantu UMKM Jamu beradaptasi dan memperluas jangkauan pasar.

### 2. Memperluas Wawasan dan Literasi Jamu
* **Peran Edukasi:** "Bae" tidak hanya berfungsi sebagai *marketplace*, tetapi juga sebagai sumber edukasi tepercaya.
* **Fokus:** Menyediakan informasi detail mengenai khasiat setiap bahan alami, mengenalkan berbagai resep dan modifikasi Jamu, serta membantu pengguna memilih Jamu yang tepat (personalisasi) berdasarkan keluhan atau kebutuhan kesehatan mereka.

## ğŸš€ Panduan Instalasi Lokal

Proyek ini dikembangkan menggunakan **React Native** dan **Firebase**. Pastikan Anda telah menyiapkan lingkungan pengembangan *mobile* yang sesuai.

### ğŸ“‹ Prasyarat

* **Node.js** & **npm/Yarn**
* **React Native CLI** (atau *Expo CLI* jika menggunakan Expo)
* Lingkungan Pengembangan Android (**Android Studio**) atau iOS (**Xcode** di macOS)
* Akses ke **Konsol Firebase** dan file konfigurasi proyek (`google-services.json` atau `GoogleService-Info.plist`).

### Langkah-langkah

1.  **Kloning Repositori:**
    ```bash
    git clone [https://github.com/MyZenaa/pgpbl-jamubae.git](https://github.com/MyZenaa/pgpbl-jamubae.git)
    cd pgpbl-jamubae
    ```

2.  **Konfigurasi Firebase:**
    * Letakkan file konfigurasi Firebase yang relevan (`google-services.json` untuk Android, atau `GoogleService-Info.plist` untuk iOS) di direktori yang sesuai.

3.  **Instalasi Dependensi:**
    ```bash
    npm install  # atau yarn install
    ```

4.  **Menjalankan Aplikasi (Android):**
    ```bash
    npx react-native run-android
    ```

5.  **Menjalankan Aplikasi (iOS - hanya untuk Mac):**
    ```bash
    cd ios
    pod install
    cd ..
    npx react-native run-ios
    ```
